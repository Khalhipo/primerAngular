Con las tuberías o pipes podemos filtrar los datos a mostrar

    Convertir a mayúsculas:
  • {{usuario |uppercase }}
    • Convertir a minúsculas:
  • {{usuario |lowercase }}
    • La primera letra de cada palabra en mayúsculas:
  • {{texto |titlecase}}
    • Fecha:
  • {{fecha |date}}
    • Fecha editada:
  • {{fecha |date:"dd/MM/yyyy"}}
    • Hora:
  • {{fecha |date: "hh:mm:ss"}}
    • Dinero:
  • {{dinero |currency: "EUR"}}
    • Número redondeado:
  • {{numero_pi |number: ".3"}}
    • Seleccionar elementos de un array:
  • {{preposiciones |slice:2:5}}
    • Convertir objeto a formato json:
  • {{usuarios |json}}

Tuberías personalizadas

    El cuadrado del 5 es
  • {{5|nums: "cuadrado" }}
    • La raíz cuadrada de 81 es
  • {{81|nums: "raiz" }}
    • El número 6 en letras es:
  • {{6|nums: "aletras"}}
